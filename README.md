# Digital-Share
managing-system for 500+ harbor shares. It tracks current and past owners, lets anyone verify share validity, and enables harbor admins to move, add, or remove shares with ease -- all in a clean, user-friendly design. Build in PHP, JavaScript (React) and MySQL and works on any webserver with a db.
